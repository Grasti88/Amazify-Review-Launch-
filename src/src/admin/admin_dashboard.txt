ARL Admin Dashboard & Hotkeys

Purpose
Provide administrators control over the ARL system and quick access to key functions.

Features
--------
- View workflow status
- Monitor error reports
- Manage templates and exports
- Toggle cloud AI for full users
- Enable/disable platforms (YouTube, TikTok, etc.)
- Hotkeys:
  - H: Open Admin Dashboard
  - E: Open Error Log
  - T: Toggle AI Mode
  - S: Start Social Export
